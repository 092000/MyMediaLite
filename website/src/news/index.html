MyMediaLite 0.04 has been released

October 20, 2010. The new release ships many bug fixes, improvements, and some new features:

  • Support for recommender engines that use relations over users and/or items. To demonstrate the new interfaces, we implemented SocialMF, based on the ACM RecSys 2010 Best Paper by Jamali and Ester.
  • Additional measure for item prediction evaluation: mean average precision (MAP)
See the Changes file for more details. Download.
