MyMediaLite 0.04 has been released

October 21, 2010. The new release contains many bug fixes, improvements, and some new features:

  • Support for recommender engines that use relations over users and/or items. To demonstrate the new interfaces, we implemented SocialMF, based on the ACM RecSys 2010 Best Paper by Jamali and Ester.
  • Additional measure for item prediction evaluation: mean average precision (MAP)
See the Changes file for more details. Download it.

MyMediaLite successfully used in live evaluation

September 30, 2010. MyMediaLite developers [% INCLUDE link link='zeno' %] and [% INCLUDE link link='steffen' %], supervised by Lars Schmidt-Thieme, received two awards at the Context-aware Movie Recommendation Challenge (CAMRa), which was held at the ACM Conference on Recommender Systems in Barcelona.

Goal of the challenge was to recommend individual movies for visitors of the MoviePilot website in a certain period, e.g. the Christmas week. In the 'Live Evaluation' track, the quality of the recommendations was judged by MoviePilot users after watching the movies. This is in contrast to the usual evaluation in such challenges, which is often based on simulated users and records of past user behavior.

The method used for the 'Live Evaluation' track was the MyMediaLite engine BPR-MF, matrix factorization optimized for Bayesian Personalized Ranking, described in this UAI 2009 paper.
