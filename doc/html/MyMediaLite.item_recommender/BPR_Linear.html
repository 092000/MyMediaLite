BPR_Linear Class

Linear model optimized for BPR. Zeno Gantner, Lucas Drumond, Christoph Freudenthaler, Steffen Rendle, Lars Schmidt-Thieme (2010): Learning Attribute-to-Feature Mappings for Cold-start Recommendations in IEEE International Conference on Data Mining (ICDM 2010), Sydney, Australia. This engine does not support online updates.
