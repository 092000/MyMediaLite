RatingData Class

Documentation for this section has not yet been entered.

public class RatingData

⊟ Remarks

Data storage for rating data. The rating events are accessible in user-wise, item-wise and unsorted triple-wise order. In order to save memory, the object initially stores only the unsorted ratings. If at some point user or item-wise access is needed, the underlying data structures are transparently created on-the-fly.
