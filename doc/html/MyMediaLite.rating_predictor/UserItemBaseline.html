UserItemBaseline Class

Uses the average rating value, plus a regularized user and item bias for prediction. The method is described in section 2.1 of Yehuda Koren: Factor in the Neighbors: Scalable and Accurate Collaborative Filtering, Transactions on Knowledge Discovery from Data (TKDD), 2009 This engine supports online updates.
